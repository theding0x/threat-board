NY A09354 — Requires DHCR to develop a common application and web portal for certain funding, tax credits, loans, and grants for housing; requires such application to be available upon request from relevant state agencies; requires such web portal to be manageable by relevant state agencies; makes relevant provisions.

PRO UNDER REVIEW SPONSORS: 22 D · 1 R
WHY IT WAS FLAGGED

Bill establishes a streamlined application process and web portal for housing development funding, tax credits, loans, and grants, which reduces administrative barriers and facilitates increased affordable housing development by making it easier for organizations to access multiple funding sources.
